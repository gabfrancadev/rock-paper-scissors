# Rock Paper Scissors

Rock Paper Scissors is an interactive web-based game developed as part of The Odin Project curriculum. 
The project focuses on transforming a console-based script into a fully functional graphical user interface.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure
- Dynamic UI for player selection
- Real-time score tracking system
- Automated game over and reset logic
- Responsive layout for different screen sizes

## What I Learned
- DOM manipulation and selection
- Event-driven programming with JavaScript
- CSS layout techniques (Flexbox and Box Model)
- Refactoring procedural code into an interactive interface
- Version control with Git (feature branching and merging)
